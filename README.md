# Empresa de Segurança - Website Institucional

## Descrição
Este é um website institucional desenvolvido em HTML para uma empresa de segurança privada. O objetivo do site é apresentar os serviços da empresa, informações de contacto e permitir que clientes conheçam melhor as soluções de segurança oferecidas.

O projeto foi desenvolvido com foco em simplicidade, responsividade e facilidade de personalização, podendo ser utilizado por qualquer empresa ou profissional da área de segurança.

## Funcionalidades

- Página inicial moderna e profissional
- Apresentação da empresa
- Listagem de serviços de segurança
- Seção de contacto
- Design responsivo para dispositivos móveis e computadores
- Navegação simples e intuitiva

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (opcional)

## Estrutura do Projeto
index.html

## Como Utilizar

1. Faça o download ou clone o projeto.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Personalize os textos, imagens e contactos conforme a necessidade da sua empresa.

## Personalização

Você pode alterar:

- Nome da empresa
- Logotipo
- Serviços oferecidos
- Telefones e e-mails
- Endereço
- Cores e identidade visual

## Serviços Exemplos

- Segurança Patrimonial
- Vigilância Privada
- Monitoramento 24 Horas
- Controle de Acesso
- Segurança de Eventos
- Consultoria em Segurança

## Compatibilidade

O website é compatível com os principais navegadores modernos:

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Safari
